# Brain Tumor Classification

This repository contains a brain tumor image classification project built around a trained EfficientNet model.

## Contents

- `main.ipynb` - notebook for data loading, training, evaluation, and inference
- `effnet.h5` - trained model weights
- `Training/` - training images organized by class
- `Testing/` - testing images organized by class
- `logs/` - TensorBoard training and validation logs

## Classes

- glioma_tumor
- meningioma_tumor
- no_tumor
- pituitary_tumor

## Notes

The dataset is organized into separate training and testing folders, each containing the four tumor classes above.
